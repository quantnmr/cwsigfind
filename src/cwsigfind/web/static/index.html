CWSigFind — Help

What is this app?

CWSigFind is a live aggregator for CW (Morse) spots. It merges real-time activity from POTA, SOTA, WWFF, WWBOTA, multiple DX cluster nodes and (optionally) the Reverse Beacon Network into a single sortable table.

Click any row to tune your radio to that frequency and mode over Hamlib rigctld. The footer shows your rig's live VFO, and the bottom-left widget shows which NCDXF/IARU beacon is transmitting right now as a quick propagation check.

Tip: press ? any time to open this panel and Esc to close it.

Spot sources

Each source has a chip in the header — click a chip to hide that source. The colored badge below matches what you see in the Src column.

POTA Parks On The Air ⚡⚡⚡ heavy

Public park activations, dominated by North America. Polled from the official REST API.

Endpoint: https://api.pota.app/spot/activator · Chip: POTA

SOTA Summits On The Air ⚡⚡ moderate

Mountain-summit activations. Strong CW culture, lots of Europe and US west-coast summits.

Endpoint: https://api2.sota.org.uk/api/spots/-1/all · Chip: SOTA

IOTA Islands On The Air ⚡ occasional

No dedicated feed — IOTA refs like EU-005 or NA-052 are detected inside DX-cluster comments and the island-group name is looked up from a weekly cache of iota-world.org.

Surfaces under the DX chip with an IOTA badge.

WWFF Worldwide Flora & Fauna ⚡⚡⚡ heavy

Park-style activations with strong EU/UK coverage that POTA doesn't carry. Polled from the public Spotline feed.

Endpoint: https://spots.wwff.co/static/spots.json · Chip: WWFF

BOTA Bunkers On The Air (WWBOTA) ⚡⚡ moderate

Newer program (2024+), growing fast in Europe. Refs look like B/G-2453. Polled from the official API with ETag caching.

Endpoint: https://api.wwbota.org/spots/ · Chip: BOTA

DX DX cluster ⚡⚡⚡ heavy

Persistent telnet feeds from multiple cluster nodes — NC7J (general worldwide) and S50CLX (filtered to POTA/SOTA/WWFF/BOTA/HEMA tags) by default; DXMAPS is configured but off (VHF/UHF). Cluster comments are also scanned for WCA / DCI / DFCF castle refs.

Endpoints: dxc.nc7j.com:7373, s50clx.infrax.si:41112 · Chip: DX

RBN Reverse Beacon Network ⚡⚡⚡⚡ firehose

Worldwide network of CW/RTTY skimmers. Massive volume — disabled by default. Flip [sources.rbn] enabled = true in config.toml to turn it on (great during CWT).

Endpoint: telnet.reversebeacon.net:7000 · Chip: RBN

Activity programs tagged on spots

When a spot belongs to an awards program, the Activity column shows a colored program badge plus the reference.

  • POTA — Parks On The Air. Refs: K-4567, VE-1234.
  • SOTA — Summits On The Air. Refs: W6/CT-001.
  • IOTA — Islands On The Air. Refs: NA-052, EU-005. Detected from cluster comments; island name resolved from the weekly IOTA cache.
  • WWFF — Worldwide Flora & Fauna. Refs like KFF-1234, GFF-0001.
  • BOTA — Bunkers On The Air. Refs: B/G-2453.
  • WCA — Castle-family awards, grouped under one badge for UI simplicity:
    • WCA — World Castles Award.
    • DCI — Diga Italiana (Italian Castles).
    • DFCF — Diplôme Français des Châteaux Forts.
    All three are picked up from explicit keywords in cluster comments; the original prefix is preserved in the activity ref.

Using the page

Filter chips

Top of the header — source chips (POTA, SOTA, WWFF, BOTA, DX, RBN) and band chips (160m … 2m). Click any chip to toggle it. With no band chips active, all bands are shown.

Search box

Free-text filter across callsign, program, activity ref / name, country, state and comment. Type and watch the table narrow live.

Max spots dropdown

Caps the number of visible rows (20 / 50 / 100 / All) so the page stays responsive during busy periods. The full client-side history (last 1000 spots) is kept in memory regardless.

Click a row to tune

Click anywhere on a row to QSY your radio to that frequency and mode. If the rig isn't connected yet, the Radio drawer opens instead so you can connect first.

Radio drawer

Click Radio… in the header. Pick your rig model (~300 Hamlib-supported radios), serial port, optional baud rate and rigctld TCP port. Click Connect & Test — the daemon spawns rigctld and reads the rig's current VFO/mode to verify the round-trip. The successful config is saved to ~/.cwsigfind/rig.json and auto-reconnected on the next startup.

Beacons widget

Bottom-left panel: which NCDXF/IARU beacon is currently transmitting on each of the five HF beacon frequencies (14.100, 18.110, 21.150, 24.930, 28.200 MHz). The 18-station cycle takes three minutes per band; the widget polls every 10 seconds so it stays in lock-step with the schedule. Click the header to collapse/expand. Click any beacon row to tune your radio to that beacon (always CW) — useful for a quick propagation sanity-check on a given band.

Propagation panel

Bottom-right panel: live space-weather indices polled from hamqsl.com every 15 minutes server-side (the UI re-reads the cache every 5 minutes). Includes:

  • SFI — solar flux at 2800 MHz (10.7 cm). Higher is better for HF.
  • SSN — daily sunspot number from N0NBH.
  • A — planetary A-index (24-hour geomagnetic activity). Lower is better.
  • K — planetary K-index (3-hour geomagnetic activity, 0–9). 0–2 quiet, 3–4 unsettled, 5+ stormy.
  • X-ray — GOES X-ray class (A/B/C/M/X). M and X classes can briefly black out HF on the daylit side.

Below the indices: the four hamqsl band buckets (80m–40m, 30m–20m, 17m–15m, 12m–10m) coloured Good / Fair / Poor for both day and night. If the latest server-side fetch failed, the panel dims and shows a "stale" indicator while keeping the last-known values visible.

Tips

CWT mini-tests (CW Ops)

Weekly 1-hour CW sprints: Wed 1300Z / 1900Z, Thu 0300Z / 0700Z. For these windows, set [sources.rbn] enabled = true in config.toml — RBN becomes the firehose you actually want. CWT calling frequencies (kHz):

1818 · 3528 · 7028 · 10118 · 14028 · 18078 · 21028 · 24908 · 28028

Modes

The server-side mode filter defaults to CW. If you want SSB or digital modes too, edit [filter].modes in config.toml.

Multiple DX clusters

NC7J is the general worldwide spot stream; S50CLX is pre-filtered to the activity programs (POTA / SOTA / WWFF / BOTA / HEMA), which is usually what cwsigfind users want. DXMAPS is configured but disabled by default — turn it on if you do VHF/UHF CW.

Glossary

  • CW — Continuous Wave; Morse code.
  • kHz — kilohertz; how frequencies are reported in the table.
  • Spot — a real-time report that this callsign is active on this frequency right now.
  • Spotter — the station (or automated skimmer for RBN) that reported the spot.
  • Activity ref — the program-specific reference, e.g. K-4567 (POTA), W6/CT-001 (SOTA), NA-052 (IOTA), B/G-2453 (BOTA).
  • QSY — change frequency.
  • NCDXF / IARU beacons — 18 worldwide stations that take turns transmitting on five HF beacon frequencies; a quick way to check propagation.

Keyboard shortcuts

  • ? — open this help panel (only when no text input is focused).
  • Esc — close the help panel or Radio drawer.
  • Click row — QSY to that spot's frequency and mode (or open the Radio drawer if the rig isn't connected yet).
